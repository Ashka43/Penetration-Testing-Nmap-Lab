# Commands Used in This Project

## Network Configuration
- `ip a`
- `ifconfig`

## Connectivity Testing
- `ping <target_ip>`

## Nmap Scanning
- `nmap <target_ip>`
- `nmap -p- <target_ip>`
- `nmap -sV <target_ip>`
- `nmap -O <target_ip>`
- `nmap -A <target_ip>`
- `nmap --script vuln <target_ip>`

> Note: `<target_ip>` refers to the Metasploitable2 machine within a controlled lab environment.
